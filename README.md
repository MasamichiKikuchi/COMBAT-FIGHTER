# COMBAT FIGHTER（コンバットファイター）
  
## 概要説明
 専門学校入学前に独学でプログラムとUnityを勉強し、個人製作した作品です。<br>

 戦闘機を操作する３Dフライトシューティングゲームで、自分が好きなシューティングゲームをモチーフにして製作しました。​ <br>

## 開発期間  
 2023/11/23~2024/1/16<br>
  
## 開発環境：開発時に使用したツールや言語など  
 言語:C#<br>
 Unity<br>

## 必要動作環境  
Windows10/11 
キーボード＆マウス

## 起動方法  
[Releasesページ](https://github.com/MasamichiKikuchi/MagnetShooter/releases)より実行ファイルをダウンロード  
→COMBAT_FIGHTER.zipフォルダをダウンロード・解凍  
→FlightShooting.exeを実行  
  
## 終了方法  
altキー & F4キー 同時押し

## 操作説明
［ゲーム画面］<br>
 Ｄキー　　　　　　・・・・・　右ロール<br>
 Ａキー　　　　　　・・・・・　左ロール<br>
 Ｓキー　　　　　　・・・・・　機首上げ<br>
 Ｗキー　　　　　　・・・・・　機首下げ<br>
 Ｅキー　　　　　　・・・・・　右ヨー<br>
 Ｑキー　　　　　　・・・・・　左ヨー<br>
 マウス右クリック　・・・・・　ミサイル発射<br>
 マウス左クリック　・・・・・　ターゲット切り替え<br>
 マウスホイール上　・・・・・　加速<br>
 マウスホイール下　・・・・・　減速<br>
 ＳＰＡＣＥキー　　・・・・・　メニュー<br>
